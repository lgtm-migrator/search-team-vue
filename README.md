# Search Team Vue.js

> A Vue.js project that gets a list of users and has the functionality to search/filter them

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
